## Themes for Zed code editor & Wezterm

Instructions:

Clone the repo (or) download the themes/spacedust.json file.

Locate the themes directory:

  macOS and Linux:
      
      ~/.config/zed/themes/

  Windows: 
      
      `%USERPROFILE%\AppData\Roaming\Zed\themes\`

Place the file there.

Apply the theme:
Once the theme file is saved, open Zed. Your new theme should appear in the theme selector, accessible through the command palette (zed: open settings file or ctrl-alt-,) or the Settings Editor (ctrl-,). Select your custom theme to apply it.
On macOS- you can try doing (Cmd k + Cmd t), search for the theme and apply it.
